# Social Preview Design Note

## Goal

Create a social preview image for GitHub and social sharing that immediately communicates three things:

1. this repository is about AI-native product development,
2. it is methodical rather than hype-driven,
3. it is designed for humans working with coding agents.

## Visual Direction

The image should feel:

- technical,
- editorial,
- controlled,
- memorable,
- professional enough for GitHub, LinkedIn, X, and search previews.

## Core Composition

- Dark foundation to create contrast and seriousness
- Bright green accent to signal momentum and action
- Grid and connector motifs to imply harnesses, workflows, and structure
- Strong title lockup with large, legible typography
- Minimal text, because the image needs to scan well at thumbnail size

## Messaging

Primary message:

`Harness-First Agentic Development`

Secondary message:

`Humans steer. Agents execute.`

## Usage

This asset is intended for:

- GitHub social preview image
- repository sharing on X / LinkedIn
- landing page or article hero reuse
